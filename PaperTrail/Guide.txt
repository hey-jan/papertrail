#1 Overview
Design a modern, scalable, and fully responsive Online Bookstore Management System web application using an MVC architecture. The system must support both customer-facing features and an administrative dashboard, focusing on transactions, account management, profile uploads, and search functionality.

#2 Design Style
2.1 Use a clean, minimalist, and professional aesthetic similar to major e-commerce platforms.
2.2 Apply a light color scheme with neutral tones and one accent color.
2.3 Ensure consistent and readable typography.
2.4 Use card-based layouts and clear visual hierarchy.
2.5 Make the interface fully responsive across desktop, tablet, and mobile.
2.6 Include hover effects, loading states, and subtle animations.
2.7 
- Editorial
- Bookstore
- Library-inspired
- Warm neutral
- Premium soft surfaces


#3 Authentication and Account Management
3.1 Implement user registration with name, email, password, and confirmation validation.
3.2 Provide secure login and logout functionality.
3.3 Include error handling and feedback messages.
3.4 Allow users to edit personal details.
3.5 Allow password updates.
3.6 Enable profile picture upload with preview and validation.
3.7 Display user information and order history in profile.

#4 Transaction System
4.1 Allow users to add books to cart.
4.2 Support updating quantities and removing items.
4.3 Provide checkout with order summary and total calculation.
4.4 Create and store order records after checkout.
4.5 Include order statuses: pending, paid, shipped, completed.
4.6 Display transaction history in user profile.
4.7 Allow admin to manage and update transaction status.

#5 Search Functionality
5.1 Include a global search bar in the navigation.
5.2 Support autocomplete or real-time suggestions.
5.3 Display search results in a grid layout.
5.4 Provide filters: category, price range, rating.
5.5 Include sorting: relevance, price, popularity.
